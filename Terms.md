| Variable       | Description                       |
| -------------- | --------------------------------- |
| `alpha`        | learning rate                     |
| `lambda_param` | lambda parameter for regularation |
| `epochs`       | no of training loops              |
| `X` / `X_train`| Feature matrix / input data       |
| `y` / `y_train`| Target labels / ground truth      |
| `w`            | Weights coefficients              |
| `b`            | Bias / intercept term             |
| `n_samples`    | Total number of data points       |
| `n_features`   | Number of features / dimensions   |
| `k`            | Number of neighbors or clusters   |
| `centroids`    | Cluster centers (e.g. K-Means)    |
| `predictions`  | Model outputs or inferred values  |
